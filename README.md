# Bobo's Rocket Rescue 🚀👽

A fun space adventure game for Bobo! Help Bobo the astronaut pilot his rocket through space and rescue friendly aliens before they float away.

## How to Play

1. Click or tap anywhere to make the rocket fly upward
2. Rescue 10 colorful aliens floating through space
3. Avoid the gray asteroids — they cost you a life!
4. You have 3 hearts. Save all the aliens to win!

## Features

- 🚀 Simple one-tap controls perfect for little hands
- 👽 Cute colorful aliens with wiggly antennae
- ☄️ Spinning asteroids to dodge
- 🔊 Web Audio API sound effects
- 🗣️ Says "Good Game, Bobo!" at the end
- 📱 Works on phones, tablets, and computers
- 🎉 Confetti particle celebrations
